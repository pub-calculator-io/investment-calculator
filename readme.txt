This free investment calculator considers the initial and ending balances, return rate, and investment time when evaluating various investment circumstances.

[https://www.calculator.io/investment-calculator/](https://www.calculator.io/investment-calculator/)

== Usage ==

Add the Investment Calculator shortcode to your page, post or sidebar:

`[ci_investment_calculator]`

Add the following code to your template where you would like the Investment Calculator to appear:

`<?php display_ci_investment_calculator(); ?>`

== Installation ==

1. Upload the Investment Calculator /ci_investment_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Investment Calculator plugin through the "Plugins" menu in WordPress.
